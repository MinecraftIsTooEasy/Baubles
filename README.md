# Baubles

Baubles is a FishModLoader mod for MITE that adds an expandable bauble equipment system to the player inventory.

It provides 12 dedicated bauble slots, including head, amulet, back, feet, bracelet, hand, ring, belt, and charm slots. Players can equip compatible accessories through the expanded inventory interface, while mod developers can use the included Baubles API to add their own wearable items and check equipped baubles in a consistent way.

## Features

- Adds 12 bauble slots to the player inventory
- Includes an expanded inventory GUI
- Supports right-click equipping for valid baubles
- Provides a Baubles API for other mods
- Includes a dedicated Baubles creative tab
- Comes with the Miner's Ring as a bauble item
